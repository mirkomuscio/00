¿Qué es el HTML? ¿En qué consiste?

El HTML un lenguaje de marcado que le indica a los navegadores web cómo estructurar las páginas web que estás visitando. Puede ser tan complejo o tan simple como desee el desarrollador web. Este consiste en una serie de elementos, que puedes utilizar para encerrar, delimitar o marcar diferentes partes del contenido para hacer que aparezcan de una cierta manera, o actúen de determinada forma.

¿Qué es una etiqueta? ¿Funciona si se escribe tanto en mayúscula como en minúscula? ¿Cuál es la mejor práctica para escribir etiquetas?

Una etiqueta se usa para crear un elementos. Funcian en ambos casos, tanto en minuscula como en mayuscula, pero esincorrecto escribirlo en mayuscula.

¿A qué llamamos elemento HTML? ¿Cuál es su anatomía? Explicar sus partes.

El HTML es un elemento de una pagina web, este puede tener datos, texto o imagenes. Su anatomia esta formada por una etiqueda de apertura, su copntenido y una etiqueta de cierre. La etiqueta de apertura consiste en el nombre del elemento, encerrado entre parentesis angulares de apertura y cierre. Esta etiqueta de apertura consiste en el nombre del elemento entre parentesis angulares de apeertura y cierre, marcando donde comienza el elemento. El contenido es el contenido del elemento. En este ejemplo, es el texto del párrafo. La etiqueta de cierre es lo mismo que la etiqueta de apertura, excepto que incluye una barra diagonal antes del nombre del elemento. Esto indica dónde termina el elemento; en este caso, dónde finaliza el párrafo. No incluir una etiqueta de cierre es un error común de principiante, y puede conducir a extraños resultados.

¿Para qué sirve la etiqueta img? ¿Cuáles son sus atributos? ¿Para qué sirven cada uno de ellos?

La etiqueta "img" representa una imagen en un documento. Esta etiqueta tiene muchos atributos globales y los mas importantes son "alt" y "src". El atributo alt este atributo define el texto alternativo que describe la imagen, texto que los usuarios verán si la URL de la imagen es errónea o la imagen tiene un formato no soportado o si la imagen aún no se ha descargado. src es obligatorio para el elemento img. En navegadores que soportan srcset, src es tratado como imagen candidata con una densidad del píxel 1x sino una imagen estará definida en srcset o srcset contiene ancho.
